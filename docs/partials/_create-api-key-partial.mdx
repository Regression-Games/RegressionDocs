Enter a unique name for your key and then click **Generate Key**.
API keys have a default expiration date of 1 year.

:::danger
Make sure to copy the generated key and store it somewhere safe.
Once the success dialog is dismissed, you will never be able to see the generated value through the Regression Games client again!
:::
