Make any modification to your code - for now, you can keep it simple by using the following snippet to make your bot say "Hello World" when it spawns into the game.
Copy and paste this into your `index.js` file (or `start.py` if your AI is a Python project):

<Tabs>
<TabItem value="ide_start_js" label="JavaScript" default>

```javascript
// inside configureBot
bot.on('spawn', async () => {
    bot.chat('Hello World!');
});
```

</TabItem>
<TabItem value="ide_start_ts" label="TypeScript">

```typescript
// inside configureBot
bot.on('spawn', async () => {
    bot.chat('Hello World!');
});
```

</TabItem>
<TabItem value="ide_start_py" label="Python">

```python
# inside configure_bot
@On(bot, 'spawn')
def bot_on_spawn(this):
    bot.chat('Hello World!')
```
